07-菲波那切数列

题目描述

大家都知道斐波那契数列，现在要求输入一个整数n，请你输出斐波那契数列的第n项（从0开始，第0项为0）。n<=39

解法

利用传统的递归方法会有重复计算，时间复杂度也会超过限制，在这里利用三个变量进行简单的循环即可
时间复杂度：O(n),空间复杂度：O(1)

代码如下： 

class Solution {
public:
    int Fibonacci(int n) {
        int res=0;
        if(n==0||n==1){
            return n;
        }
        int one=0,two=1,three=0;
        for(int i=2;i<=n;i++){
            three=one+two;
            one=two;
            two=three;
        }
        return three;
    }
};

08-跳台阶

题目描述

一只青蛙一次可以跳上1级台阶，也可以跳上2级。求该青蛙跳上一个n级的台阶总共有多少种跳法（先后次序不同算不同的结果）。

解法

思想同上一题：菲波那切数列
时间复杂度：O(n),空间复杂度：O(1)

代码如下： 

class Solution {
public:
    int jumpFloor(int number) {
        if(number<=0){
            return 0;
        }
        else if(number <3){
            return number;
        }
        int one=1,two=2,three=0;
        for(int i=3;i<=number;i++){
            three=one + two;
            one=two;
            two=three;
        }
        return three;
    }
};
